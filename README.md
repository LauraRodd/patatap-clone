# Patatap Clone

A [Patatap](https://patatap.com/) clone built with [paper.js](http://paperjs.org/) and [howler.js](https://howlerjs.com/)


## How to use:

Press any key on your keyboard and turn up your volume.

### Technologies and tools used:

- CSS
- Paper.js
- Howler.js
